## 概要

黒潮大蛇行（LM/NLM）レジームが日本近海4魚種の資源量に与える影響を、捕食被食ODEモデルとパラメータ推定で定量化する研究コード。

- **被食者 (x)**: マイワシ (x1), カタクチイワシ (x2)
- **捕食者 (y)**: ブリ (y1), サワラ (y2)
- **レジーム分割**: NLM 2006–2016 / LM 2017–2024

### ODE推定の流れ（`現行コード/`）

1. `data_loader.load_clean_dataframe()` → 4種CSVをマージ、NaN除去
2. `data_loader.get_series()` → 全種を千トン統一、漁獲圧 `f = catch/biomass`（上限0.95）を計算
3. NLM/LMの年次マスクで時系列を分割
4. `model.estimate()` → 各レジームをODE推定
   - 正規化空間（各種を全期間平均で除した平均1.0の空間）でODEを解く
   - `solve_ivp(method="LSODA")` 積分 → `least_squares(method="trf")` で対数誤差最小化
   - マルチスタート（`n_starts`）で局所解回避、相互作用パラメータに L2 正則化（`reg_lambda`）
5. 推定後に `model._to_absolute()` で元スケールの物理パラメータへ換算

### モデル定義（`model.py`, capacity_ry 12変数）

正規化空間の ODE 右辺（`make_ode`）:
```
dx1 = (r_x1 − f_x1)·x1 − L11·x1·y1 − L12·x1·y2
dx2 = (r_x2 − f_x2)·x2 − L21·x2·y1 − L22·x2·y2
dy1 = (−r_y1 − f_y1)·y1 + C1·L11·x1·y1 + D1·L21·x2·y1
dy2 = (−r_y2 − f_y2)·y2 + C2·L12·x1·y2 + D2·L22·x2·y2
```
- `r_x1, r_x2`: 被食者の自然増殖率 / `r_y1, r_y2`: 捕食者の自然死亡率
- `L11..L22`: 捕食圧（相互作用係数） / `C1,D1,C2,D2`: 捕食→捕食者への変換効率
- **密度依存項（種内競争 α）は含まない**

> **モデル方針（2026-06-13決定）**: 環境収容力なし（種内競争項 α なし）モデルのみで進める。
> `capacity_ry`（12変数, r_y自由化）を主力、`capacity`（10変数, r_y固定）も使用可。
> `full`（16変数, 種内競争項あり）は LM 期8点に対し過剰パラメータで識別性崩壊・非物理的パラメータ（c1=41等）が生じるため**使用しない**。

### 適合度指標

- **RMSE** `sqrt(mean((obs−pred)²))` — 千トン、絶対誤差
- **NRMSE** `RMSE / mean(obs)` — 無次元、魚種横断で比較可能
- **R²** `1 − SS_res/SS_tot` — 平らなデータ（分散極小）で負に暴れるアーティファクトに注意

→ **評価はNRMSE主体、R²は補助**。

## 研究進捗ログ

### Phase 1〜4: 推定手法の確立・魚種選定・4種推定

- **Phase 1**: 単一スタート推定が局所解に陥る問題を発見。マルチスタート・LSODA積分・L2正則化で精度改善。
- **Phase 2**: 7魚種の全組み合わせをCV基準でスキャン。ブリ+サワラを捕食者に入れるとLM期の安定性改善。
- **Phase 3**: 捕食者ブリと被食者ヤリイカの生物量逆転で変換効率 c1 が非物理的（41.1）になる問題が判明。
- **→ 採用**: マイワシ(x1)＋カタクチ(x2) / ブリ(y1)＋サワラ(y2)。
- **Phase 4**: この組み合わせで c1異常値が解消（NLM=0.36, LM=0.03）。NLMは正則化不要（11点で識別性が保てる）。NRMSE水準 NLM 0.15〜0.23 / LM 0.16〜0.22。

### Phase 5: MSY計算と持続性制約（`現行コード/msy/`）

**手法**: 既存ODEに定数漁獲圧ベクトル `f` を注入し、積分期間Tの平均漁獲量を最大化する `f*` を4次元グリッド全評価（6⁴=1296点）で探索。

- **Phase 5 発見**: 戦術的MSY（T=1年）は全条件で f*=0.95 張り付きで情報が乏しい → MSYの本質は戦略的（長期T）でのみ立ち上がる。LMの戦略的MSYがNLMの約7倍で内訳97%がマイワシ → 長期外挿のアーティファクト疑い。
- **Phase 5a（非減少制約）**: `B_i(T) ≥ B_i(0)×0.9` を追加。NLMでは制約MSY≈無制約（カタクチのみ律速）、LMでは−16.7%。乱獲解（上限張り付き）を内点に引き下げられるが feasible 点が少なく精緻化が課題。
- **Phase 5b（マイワシ終端診断, `diagnose_iwashi.py`）**: LMのマイワシは f_x1=0.95 まで収量が単調増加（密度依存なしの構造的帰結）。NLMは山型（ピーク f_x1≈0.30）。→ **密度依存なし＋ブーム・バスト種はMSYが立たない**という核心的知見。

### Phase 6: MSY同定可能性の合成データ研究（構想のみ）

「どんなデータなら内点MSYを同定できるか」を合成データで体系化する構想。Phase 7 で実データ側アプローチを優先することにし、補完的位置づけに。

### Phase 7: 教授フィードバックと連続時間 Catch-MSY（2026-07-03〜）

#### 教授フィードバックの要点

1. **中立安定・自己密度依存**: 密度依存がないと漁獲量が f の1次（単調）になる点は我々の診断（Phase 5b）と一致。「自己密度依存なし＋環境変動」の中立安定を扱う題材として本モデルを肯定的に評価。
2. **単一種モデルで増殖率を外挿値として固定する提案**（採用）:
   - `r_x1, r_x2`（被食者）はそのまま使える。
   - 捕食者の増殖率は餌密度依存で変動するが、**最大値として `c1+d1, c2+d2`**（変換効率＝増加率/単位捕食量の和）を推定できる。根拠: `(1/y1)dy1/dt = −r_y1 + c1·(x1捕食量) + d1·(x2捕食量)`、各餌を最大限（単位捕食量=1）食べたときの増殖率上限が `c1+d1`。
   - 具体策: **Catch-MSY法**で `r_x1, r_x2, c1+d1, c2+d2` を先に推定 → 外挿値固定 → 残りパラメータ推定 → MSY導出。
3. **データ**: 推定資源量の長期時系列は存在しない。長期に存在するのは漁獲量（農水省統計, 1956〜）。純粋なCatch-MSY（漁獲量のみ、資源量不使用）で進め、終端枯渇度の感度分析をセットで用意。

#### 連続時間 Catch-MSY の実装（`現行コード/catch_msy/`, 完了）

- 古典（離散）`B_{t+1}=B_t+r·B_t(1−B_t/K)−C_t` でなく、**連続** `dB/dt=r·B(1−B/K)−C(t)` を `solve_ivp` で解く（C(t)は年次漁獲量の線形補間）。
- Martell & Froese リサンプリング法（連続版）: 事前分布(r,K,B0)からサンプル → 各ペアで積分 → 全期間(0,K]内かつ終端枯渇度がレンジ内のペアだけ残す → 生存 r からMSY=rK/4。
- **速度設計**: N成分を「独立だが連立した1本のODE系」として1回の `solve_ivp` でベクトル化積分（`integrate_batch`）→ ~0.5秒/種（1ペアずつ solve_ivp の~15分/種から3000倍高速化）。崩壊は右辺 `B=max(B,0)` クランプ→`min_t B(t)≤0` で判定（events版と棄却セマンティクス同一、500ペアで100%一致検証済）。

#### 終端枯渇度レンジの設定（Martell & Froese 2013 Table 1）

「終端年 catch/max catch 比」で B/k 事前レンジを場合分け（`RESILIENCE_R` は同論文 Table 2 と一致）:

| | Catch/max catch | B/k レンジ |
|---|---|---|
| First year | <0.5 / ≥0.5 | 0.5–0.9 / 0.3–0.6 |
| Final year | >0.5 / ≤0.5 | 0.3–0.7 / 0.01–0.4 |

原論文 Caveats: *"assumes a stationary production function... no change in the parameters over time"*。レジームシフト型ブーム・バスト種はこの定常性前提が破れる。

#### Phase 7a/7b: 同定可能性の検証結果（全国版データ, 1956-2024）

**核心結論**: 連続時間Catch-MSYで実用的に解けるのは **ブリ・サワラ（捕食者）のみ**。

|---|---|---|---|---|
| ブリ | n=516, r=0.345 | n=1289 | n=11266, r=0.446 | ✅ 全レンジ可 |
| サワラ | n=238, r=0.280 | n=822 | n=10692, r=0.430 | ✅ 全レンジ可 |
| マイワシ | n=0 | n=0 | n=274, r=0.728 | ⚠ 高レンジのみ |
| カタクチ | n=0 | n=2 | n=3407, r=0.911 | ⚠ 高レンジのみ |
| サバ類 | n=0 | n=2 | n=4625, r=0.397 | ⚠ 高レンジのみ |
| スルメイカ | n=0 | n=0 | **n=0** | ✕ 全滅（B0拡張でも0） |
| スケトウダラ | n=0 | n=0 | n=7 | ✕ 実質不可 |

- **ブリ・サワラ**: 原論文標準ルール（`[0.3,0.7]`/`[0.01,0.4]`）で確定値。**`c1+d1≈0.38`（ブリ）, `c2+d2≈0.28`（サワラ）を採用可**。
- **マイワシ・カタクチ・サバ類**: ブーム・バストで高レンジ[0.6,0.95]（＝現在高水準前提）でしか解けない。
- **スルメイカ**: 深い枯渇（終端/max=0.03）＋多周期＋寿命1年の年級群卓越型で全滅。
- **スケトウダラ**: 単峰型でも 1956→1972 で約13倍という**急峻な立ち上がり**がロジスティックの表現力を超え、実質不可（「単峰なら解ける」の予想は外れた）。

#### Phase 7c: 系群混在の検証とデータ方針転換（2026-07-04）

**動機**: 現行ODE推定の資源評価データは「太平洋系群」限定（ファイル名 `カタクチイワシ太平洋時系列データ`）。一方 Catch-MSY で使っていた e-stat は「全国」合算。系群混在がカタクチ等の同定困難の一因ではという仮説を、太平洋12県合算データで検証。

**結果（太平洋12県 vs 全国, カタクチ/max比較）**: 全国合算に対する太平洋12県の比率は年により20〜72%と大きく変動 → **別系群の異なるタイミングの変動が混ざって多峰化していた**ことを裏付け。

| 魚種 | 太平洋12県 [0.1,0.4] | 全国 [0.1,0.4] | 効果 |
|---|---|---|---|
| カタクチ | **n=14, r=0.70** | n=0 | ✅ 低レンジで解けるように改善 |
| サバ類 | n=5, r=0.25 | n=0 | ○ 改善 |
| ウルメイワシ | n=1093, r=0.93 | （全国 n=114） | ◎ 全レンジ堅牢 |
| マイワシ | n=0（全滅） | n=0（[0.6,0.95]でn=274） | ✕ **逆に悪化** |

- **カタクチは仮説通り改善**: 太平洋限定で「終端は低い」という素直な解釈とモデル可解性が初めて両立。
- **マイワシは逆に悪化**: 太平洋系群こそ教科書的レジームシフト魚種の本体で、全国合算はむしろブーム・バストを平滑化していた。太平洋限定は全レンジ n=0。
- **含意**: 系群混在はカタクチ同定困難の一因と実証されたが万能薬ではない。マイワシは太平洋系群では別手法が必要。

**カタクチの代替候補としてウルメイワシを追加**: マイワシ属近縁・黒潮沿岸分布・ブリの餌生物で生態的に自然。緩やかな定常変動でCatch-MSYが最も得意な形。全終端レンジで堅牢に解ける（Table1該当[0.4,0.7]で r≈0.75）。ローダ key=`urume`（resilience=high）。

**2016-2023補完後の再検証（太平洋12県, 1956-2023）**: 終端年が2015→2023に延びて結果が動いた。**カタクチは再び困難化**（[0.1,0.4] n=0, [0.6,0.95]でもn=21のみ）——2015打ち切り版で「太平洋限定なら低レンジで解ける（n=14）」とした改善は**2015終端の部分的アーティファクト**で、2016-2023の再減少（終端/max=0.09）を加えると多峰性が復活する。一方 **ブリ・サワラ・ウルメイワシは全レンジで堅牢**（ブリr≈0.37-0.45, サワラr≈0.38-0.44, ウルメr≈0.75-0.93）。マイワシは[0.6,0.95]のみ（r≈0.90, 終端/max=0.20）、スルメイカ全滅・スケトウダラ実質不可は不変。→ **カタクチのウルメイワシ置換がより強く正当化された**。

#### 描画バグの修正（解決済み）

推定フィット図がカクカクだった原因は、ODEは連続で解いているが軌道を年次点だけでサンプリングして直線接続していたため（`model.simulate` の `t_eval=t_rel`）。`現行コード/msy/plot_fit_smooth.py` で、推定後にパラメータ固定のまま細かい時間グリッド（`linspace(t0,tend,300)`）で再積分し滑らかな1本に。推定ロジック・R²は非変更。

### Phase 7d: 種構成の確定とウルメイワシ資源評価データの導入（2026-07-07）

#### 種構成を確定

**被食者: マイワシ(x1) ＋ ウルメイワシ(x2, カタクチイワシから置換) ／ 捕食者: ブリ(y1) ＋ サワラ(y2)** で一旦確定。

`現行コード/catch_msy/catch_data_loader.py` の `MAIN_KEYS` を `["sardine", "urume", "buri", "sawara"]` に変更。`run_catch_msy.py` に魚種ごとの終端枯渇度レンジ `FINAL_RANGE_BY_SPECIES` を追加（Froese Table1標準ルールを種ごとに個別適用。マイワシのみ標準ルールでn=0のため例外的に高レンジ採用）。

#### ウルメイワシの資源量指標値データの発見（Notion経由）

ODE推定の状態変数x2をウルメイワシに置換するには「資源量」の時系列が必要だが、他3種と異なりウルメイワシには水産研究・教育機構の絶対資源量（トン）CSVが存在しない（e-stat漁獲量のみ）。

Notion「研究(MSY)」DBに登録されていたFRA「令和7（2025）年度ウルメイワシ太平洋系群の資源評価」詳細版PDF（[https://abchan.fra.go.jp/wpt/wp-content/uploads/2026/03/details_2025_21.pdf](https://abchan.fra.go.jp/wpt/wp-content/uploads/2026/03/details_2025_21.pdf)）から、表4-1「資源量指数および資源量指標値の相対値の推移」（1979-2024, 46年分）を抽出し `data/ウルメイワシ資源量指標値_FRA資源評価2025.csv` として保存。

この「資源量指標値」は絶対資源量ではなく、**平衡状態を仮定しない余剰生産モデルSPiCT**（Pedersen and Berg 2017）で推定した資源量の相対値（平均を1として規格化）。VPAによる絶対資源量推定は令和3年度評価で打ち切られている。

#### 漁獲圧 f_x2 の導出方法

指標値の定義 `指標値(t) = B(t)/B̄`（B̄は1979-2024年の平均絶対資源量）を `B(t)` について解き `B(t) = B̄ × 指標値(t)` とし、漁獲圧の定義 `f_x2(t) = C(t)/B(t)` に代入すると

```
f_x2(t) = C(t) / (B̄ × 指標値(t))
```

未知数は B̄（1個のスカラー）のみ。PDF補足表4-1のSPiCT推定結果（K, bkfrac=B(1979)/K, 3モデル分）から `B̄ = bkfrac×K / 指標値(1979)` で逆算し、3モデル平均で **B̄ ≈ 172千トン** と算出（K自体は3モデルで3.77e5〜4.51e5トンとばらつくが、B̄は166〜177千トンで±3%と頑健——KとbkfracがB(1979)一定の制約下で逆相関に動くため）。

結果: `f_x2` は平均0.10、範囲0.04〜0.15/年で、r_x2=0.78との整合性も良好（常に `r_x2 > f_x2` が成立し、モデルが破綻しない）。
