## 概要

黒潮大蛇行（LM/NLM）レジームが日本近海4魚種の資源量に与える影響を、捕食被食ODEモデルとパラメータ推定で定量化する研究コード。

- **被食者 (x)**: マイワシ (x1), カタクチイワシ (x2)
- **捕食者 (y)**: ブリ (y1), サワラ (y2)
- **レジーム分割**: NLM 2006–2016 / LM 2017–2024
- **旧捕食者**: ヤリイカ・スルメイカ（Phase 2 で変更済み）

## 実行コマンド

```bash
# 標準推定（現行コード/ から実行、capacity_ry 12変数）
cd 現行コード && python3 data_loader.py     # データ確認

# MSY 計算（現行コード/msy/ から実行）
cd 現行コード/msy && python3 run_msy.py            # capacity_ry（デフォルト）
cd 現行コード/msy && python3 run_msy.py capacity   # capacity
cd 現行コード/msy && python3 diagnose_iwashi.py    # マイワシ終端挙動の診断
```

出力 PNG は各スクリプトのディレクトリ（`現行コード/msy/`）に保存される。

### 推定の流れ（`現行コード/`）

1. `data_loader.load_clean_dataframe()` → 4種CSVをマージ、NaN除去
2. `data_loader.get_series()` → 全種を千トン統一、漁獲圧 `f = catch/biomass`（上限0.95）を計算
3. NLM/LMの年次マスクで時系列を分割
4. `model.estimate()` → 各レジームをODE推定
   - 正規化空間（各種を全期間平均で除した平均1.0の空間）でODEを解く
   - `scipy.integrate.solve_ivp(method="LSODA")` で数値積分
   - `scipy.optimize.least_squares(method="trf")` で対数誤差を最小化
   - マルチスタート（`n_starts`）で局所解回避、相互作用パラメータに L2 正則化（`reg_lambda`）
5. 推定後に `model._to_absolute()` で元スケールの物理パラメータへ換算

### モデル定義（`model.py`, capacity_ry 12変数）

正規化空間の ODE 右辺（`make_ode`）:
```
dx1 = (r_x1 − f_x1)·x1 − L11·x1·y1 − L12·x1·y2
dx2 = (r_x2 − f_x2)·x2 − L21·x2·y1 − L22·x2·y2
dy1 = (−r_y1 − f_y1)·y1 + C1·L11·x1·y1 + D1·L21·x2·y1
dy2 = (−r_y2 − f_y2)·y2 + C2·L12·x1·y2 + D2·L22·x2·y2
```
- `r_x1, r_x2`: 被食者の自然増殖率 / `r_y1, r_y2`: 捕食者の自然死亡率
- `L11..L22`: 捕食圧（相互作用係数） / `C1,D1,C2,D2`: 捕食→捕食者への変換効率
- **密度依存項（種内競争 α）は含まない**

> **モデル方針（2026-06-13決定）**: **環境収容力なし（種内競争項 α を含まない）モデルのみ**で進める。
> `capacity_ry`（12変数, r_y自由化）を主力、`capacity`（10変数, r_y固定）も使用可。
> `full`（16変数, 種内競争項あり）は LM 期 8点に対して過剰パラメータ・識別性崩壊・非物理的パラメータ（c1=41等）が生じるため**使用しない**。

### データスケール換算

各CSVで単位が異なるため `data_loader.get_series()` で千トンに統一：
- マイワシ: 万トン × 10 / カタクチ: 千トン（漁獲量はトン ÷ 1000）
- ヤリイカ: トン ÷ 1000 / スルメイカ: 千トン

### 適合度指標

- **RMSE** `sqrt(mean((obs−pred)²))` — 千トン、絶対誤差
- **NRMSE** `RMSE / mean(obs)` — 無次元、魚種横断で比較可能
- **R²** `1 − SS_res/SS_tot` — 平らなデータで負に暴れるので要注意

→ **評価はNRMSE主体、R²は補助**。

## 研究進捗ログ

### Phase 1〜3: 推定手法の確立と魚種組み合わせの選定

- **Phase 1**: 単一スタート推定が局所解に陥ることを発見。マルチスタート・LSODA積分・L2正則化を導入し精度改善。
- **Phase 2**: 7魚種の全組み合わせを CV 基準でスキャン。ブリ+サワラを捕食者に入れると LM 期の安定性が改善（1位: ヤリイカ+スルメイカ/ブリ+サワラ、4位: マイワシ+カタクチ/ブリ+サワラ）。
- **Phase 3**: 1位組み合わせは捕食者ブリと被食者ヤリイカの生物量逆転で変換効率 c1 が非物理的（41.1）になる問題が判明。

→ **採用組み合わせ**: マイワシ(x1)＋カタクチ(x2) / ブリ(y1)＋サワラ(y2)（Phase 4 以降）

### Phase 4: 4位組み合わせの推定（capacity_ry 12変数）

- **c1異常値が解消**（NLM=0.36, LM=0.03）: 被食者の生物量が捕食者を上回り変換効率が物理的に合理的に。
- **NLMは正則化不要**（11点・12変数で識別性が保てる。reg_lambda=0.01は効き過ぎ）。
- NRMSE水準: NLM 0.15〜0.23 / LM 0.16〜0.22。R²の負値は「平らなデータでR²が壊れる」アーティファクトでNRMSEとは矛盾しない。

### Phase 5: MSY（最大持続生産量）計算（`現行コード/msy/`）

**手法**: 既存 ODE に定数漁獲圧ベクトル `f=[f_x1,f_x2,f_y1,f_y2]` を注入し、積分期間 T の平均漁獲量（台形則/T）を最大化する `f*` を **4次元グリッド全評価（6⁴=1296点）** で探索。`make_ode()` に定数callableを渡して再利用。

- **戦略的MSY**: レジーム全期間（NLM T=10年 / LM T=7年）を1本の軌道で評価。
- **戦術的MSY**: T=1年ごとに各年の初期値から評価。

**発見**:
1. **戦術的MSYは全条件で f*=0.95 に張り付き情報が乏しい** → MSYの本質は戦略的（長期T）でのみ立ち上がる。
2. **LMの戦略的MSYがNLMの約7倍（13,792 vs 1,914 千トン/年）** で内訳の97%がマイワシ → **長期外挿のアーティファクト疑い**。
3. 捕食者ブリ・サワラの f* も上限張り付き傾向（被食者ダイナミクスへの寄与が小さく「獲るほど良い」になりやすい）。

### Phase 5a: 非減少資源量制約（持続性制約）

無制約MSYの「乱獲解（f*=0.95張り付き）」を解消するため、**期末資源量が初期値の90%以上（`B_i(T) ≥ B_i(0)×0.9`）** の非減少制約を追加（4種・endpoint判定・tol=0.1、N_GRID=8）。

- **NLMでは制約MSY ≈ 無制約（−2.1%）**: カタクチだけが初期値割れギリギリで律速、他3種は増える余裕があり制約がほぼ効かない。
- **LMでは制約MSYが顕著に低下（−16.7%）**: カタクチ枯渇を避けるため f_x2 を削減・f_y2 を増加させてバランス。
- 無制約の上限張り付きは内点に引き下げられ乱獲解を回避。ただし feasible 点が少なく（6・54件）、精緻化（2段階グリッド等）が課題。

### Phase 5b: マイワシ終端挙動の診断（`diagnose_iwashi.py`）

- **「B(T)/B(T−1) ≥ 0.9」制約は機能しない**: LV型振動系では1ステップ変化が常に小さく、高漁獲率でも比が0.9を下回らない。
- **LMのマイワシは f_x1=0.95 まで収量が単調増加**: 密度依存がないため漁獲率を上げるほど収量が増える。**密度依存なしモデルの構造的帰結**（制約設計の問題ではない）。
- NLMのマイワシ収量は山型（ピーク f_x1≈0.30）: 捕食者との相互作用で振動し、高漁獲率では資源・収量とも低下。

> **含意**: LMのマイワシ f* 上限張り付きを解消するには、(1)密度依存項の導入か、(2)「密度依存なしモデルの構造的帰結」として論文に明示、のいずれかが必要。

### Phase 6: MSY 同定可能性の条件探索（合成データ研究）（2026-06-28）

**動機**: Phase 5b より、MSYが意味を持つには(1)密度依存か(2)捕食者による十分な抑制が必要。「このモデルでどんなデータなら内点MSYを同定できるか」を合成データで体系化する構想。操作変数は密度依存強度α・捕食圧L・資源量トレンド・データ点数n。

> **状態**: 構想のみ。Phase 7 の教授フィードバックを受け、実データ側からのアプローチ（Catch-MSYで増殖率を外挿値として固定）を優先することに。合成データ研究は補完的に位置づけ。

### Phase 7: 教授フィードバックと連続時間 Catch-MSY への方針転換（2026-07-03）

#### 教授フィードバックの要点

1. **中立安定・自己密度依存**: 密度依存がないと漁獲量が f の2次（山型）でなく1次（単調）になる点は我々の診断（Phase 5b）と一致。教授は「自己密度依存なし＋環境変動」の中立安定の状況を検討する題材として本モデルを肯定的に評価。密度依存ありバージョンは別途検討の余地。
2. **データ不足**: 捕食-被食パラメータは推定値は出るが精度検証が困難。
3. **単一種モデルで増殖率を外挿値として固定する提案**（採用）:
   - 4魚種それぞれ単一種モデルを作り増殖率を推定 → 既知値として固定。
   - `r_x1, r_x2`（被食者）はそのまま使える。
   - 捕食者の増殖率は餌密度依存で変動するが、**最大値として `c1+d1, c2+d2`**（変換効率＝増加率/単位捕食量、餌を最大限食べたときの増殖率の和）を推定できる。
   - 具体策: **Catch-MSY法**で `r_x1, r_x2, c1+d1, c2+d2` を先に推定 → 外挿値固定 → 残りパラメータを推定。
4. **資源評価データの制約**: 推定資源量の長期時系列は存在しない（CPUEが精度よく取れる近年からしか資源評価しない）。実データとして長期に存在するのは**漁獲量**（農水省 農林水産統計、昭和31年=1956〜）。
5. **魚種候補（教授の主観）**: マイワシ・ブリ・スルメイカ・サバ類。魚種選定自体にデータ根拠があると良い。各魚種を4グループに分ける案も。

#### 「捕食者の最大増殖率 = c1+d1」の根拠（確認済み）

捕食者y1の per-capita 増殖率は `(1/y1)dy1/dt = −r_y1 + c1·(x1捕食量) + d1·(x2捕食量)`。`c,d` は「増加率/単位捕食量」なので、各餌を最大限（単位捕食量=1）食べたときの増殖率がそれぞれ c1, d1、2種同時なら和 `c1+d1` が理論上限。

#### 決定した方針

**パイプライン**: Catch-MSYで増殖率（r_x1,2 と c1+d1, c2+d2）を推定 → パラメータ空間を絞る → 残りパラメータ推定 → MSY導出。

**連続時間 Catch-MSY**（差分方程式でなく微分方程式で実装する — ユーザー要望）:
- 古典（離散）: `B_{t+1} = B_t + r·B_t·(1 − B_t/K) − C_t`
- **今回（連続）**: `dB/dt = r·B·(1 − B/K) − C(t)` を `solve_ivp` で解く。C(t)は年次漁獲量を補間した連続関数。
- Martell & Froese のリサンプリング法を連続時間で: 事前分布（r, K, 初期・終端の枯渇度 B/K）から (r,K) を多数サンプル → 各ペアで連続ODEを積分 → 軌道が全期間 (0,K] に収まり終端枯渇度が想定範囲に入るペアだけ残す → 生存 r 分布から推定。MSY = rK/4。

**決定事項（2026-07-03）**:
- **データ**: 農水省統計（1956-）を先に取得してから実装（現行CSVの漁獲量はマイワシ1975- / カタクチ1978- / サワラ1987- / ブリ1994-）。
- **手法**: 純粋なCatch-MSY（漁獲量のみ、資源量は使わない）。資源量を捨てる分、終端枯渇度は Froese ヒューリスティックで置くため、**終端枯渇度の感度分析をセットで用意**する（結果を最も左右するため）。

#### 次のステップ

1. **データ取得**: e-stat 農林水産統計から4魚種（＋候補のサバ類・スルメイカ）の1956-漁獲量を取得・整形し `data/` にCSV化。魚種粒度（例「まいわし」単独か「いわし類」一括か）・海域区分が現行資源評価データと揃うかが勘所。
2. **実装**: 連続時間Catch-MSY（単一種4本）。
3. **出力**: r_x1, r_x2, c1+d1, c2+d2 の推定値＋不確実性＋終端枯渇度の感度分析。

### Phase 7a: データ取得＋連続時間 Catch-MSY 実装（2026-07-03, 完了）

**データ取得（完了）**: e-stat「海面漁業魚種別漁獲量累年統計（全国）」（表番号3, 昭和31〜令和6年=**1956-2024, 69年欠損なし**）を取得。
- 保存: `data/estat_海面漁業魚種別漁獲量_全国_1956-2024.csv`（整形済, 単位トン）＋ `data/estat_..._原本.xlsx`
- 6魚種: まいわし・かたくちいわし・さば類・ぶり類・さわら類・するめいか（教授候補のサバ類・スルメイカも収録）。粒度は「まいわし」単独で取れた（「いわし類」一括ではない）。
- 注意: ブリだけ既存資源評価CSV(2024=104,298t)がe-stat全国「ぶり類」(81,404t)を上回る。定義差の可能性（Catch-MSYはe-stat系列を独立に使うので影響なし）。

**実装（完了）**: `現行コード/catch_msy/`

- `catch_msy_core.py`: 連続 Schaefer `dB/dt=r·B(1−B/K)−C(t)`、C(t)は年次漁獲量の線形補間、`solve_ivp`

**速度設計（重要）**: 1ペアずつ `solve_ivp(events,terminal=True)` で回すと30,000ペア×68年積分で**~15分/種**（7.45ms/積分）。→ **N成分を「独立だが連立した1本のODE系」として1回の solve_ivp でベクトル化積分**（`integrate_batch`）し**~0.5秒/種**（3000倍）。崩壊は右辺で `B=max(B,0)` クランプ（吸収境界）→ `min_t B(t)≤0` で判定、events版と棄却セマンティクス同一（検証: 500ペアで生存判定100%一致）。events版 `integrate_one` は参考実装として残置。

**結果（30,000サンプル, 終端レンジ感度）**:

| 魚種 | resil | [0.1,0.4] | [0.2,0.6] | [0.4,0.7] | [0.6,0.95] |
|---|---|---|---|---|---|
| マイワシ | high | n=0 | n=0 | n=0 | **0.728** (n=274) |
| カタクチ | high | n=0 | 0.695(n=2) | 0.680(n=3) | **0.911** (n=3407) |
| ブリ | medium | 0.345(516) | 0.359(1289) | 0.388(1846) | 0.446(11266) |
| サワラ | medium | 0.280(238) | 0.306(822) | 0.334(1377) | 0.430(10692) |

（値=生存 r の幾何平均, 単位1/年）

**発見**:
1. **ブリ・サワラ（捕食者）は全終端レンジで r 同定可**（r≈0.28〜0.45）。解釈Aで `c1+d1, c2+d2` の実効推定値に使える。終端レンジが高いほど r は上振れ（0.35→0.45程度）。
2. **マイワシ・カタクチ（浮魚）は高終端レンジ [0.6,0.95] でしか生存点が出ない**。ブーム・バスト＋現在の資源回復で終端資源が K 近くに張り付くため。→ 現在高水準という前提でのみ r≈0.73/0.91 と推定。低終端レンジは非該当（＝「現在枯渇」の仮定と両立しない）。これは Phase 5b の「密度依存なし＋ブーム・バスト種は同定困難」と整合する構造的帰結。

**終端枯渇度レンジの妥当な設定方法（原論文 Table 1 で確認・訂正済み）**:

Martell & Froese (2013, Fish and Fisheries 14(4):504–514) Table 1 は「終端年の catch/max catch 比」で B/k 事前レンジを場合分けする（Table 2 の resilience別 r レンジは本実装の `RESILIENCE_R` と一致）:

| | Catch/max catch | B/k レンジ |
|---|---|---|
| First year | <0.5 | 0.5–0.9 |
| | ≥0.5 | 0.3–0.6 |
| Final year | >0.5 | 0.3–0.7 |
| | ≤0.5 | 0.01–0.4 |

4種の終端年(2024)/最大漁獲量比とTable 1適用結果:

| 魚種 | 終端/max | 該当レンジ | 実行結果 |
|---|---|---|---|
| マイワシ | 0.15 | [0.01,0.4] | **n=0（生存ペアなし）** |
| カタクチ | 0.19 | [0.01,0.4] | **n=0** |
| ブリ | 0.65 | [0.3,0.7] | n=2087, r=0.384, MSY=99 |
| サワラ | 0.24 | [0.01,0.4] | n=251, r=0.278, MSY=12 |

→ **ブリ・サワラは原論文の標準ルールで問題なく解ける。マイワシ・カタクチは標準ルールでも解が出ない。**

原因は原論文自身が Caveats 節で明記する前提: *"the Catch-MSY approach also assumes a stationary production function... no change in the parameters of the Schaefer model over time"*。マイワシ・カタクチは1980年代ピーク→崩壊→近年再増加という**レジームシフト型のブーム・バスト史**を持ち、この定常性の前提が成り立たない。Table 1 の「catch/max catio」ヒューリスティックは単峰型の乱獲履歴を想定しており、複数周期の履歴には対応できない。

**結論**: マイワシ・カタクチへのCatch-MSY適用は、終端レンジを恣意的に操作して解を出すのではなく、**「このモデル（Catch-MSYの定常性前提）では同定困難」と扱うのが妥当**（Phase 5b/6の「密度依存なし＋ブーム・バスト種は同定困難」という既存知見と整合）。ブリ・サワラの `c1+d1, c2+d2` は原論文Table 1のレンジ（`[0.3,0.7]`, `[0.01,0.4]`）で確定値として採用可能。マイワシ・カタクチの `r_x1, r_x2` はCatch-MSY以外の方法（例: 単一種でも定常性を仮定しない別モデル、または教授との相談）を検討する必要がある。

```bash

cd 現行コード/catch_msy && python3 sensitivity.py      # 終端レンジ感度・箱ひげ
```
