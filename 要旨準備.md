# 要旨準備（数理生物学会 2026, 9/7–10 / 要旨締切 7/23）

> 方針＝**道A**: 「レジーム別MSF計算の枠組みを提案し、その識別性上の限界と対処を論じる」。
> 動いているパイプラインの提案（道B）に賭けず、**単一種Catch-MSY推定値を多種ODEに結合する際の構造的障害を診断すること自体を貢献にする**。9月までに障害が解ければ格上げ、解けなくても発表は成立する二段構え。

---

## 1. 発表の核（何を主張するか）

**黒潮レジーム（大蛇行 LM / 非大蛇行 NLM）で分けた最大持続生産量（MSF）を、多種捕食被食ODE ＋ 単一種Catch-MSYの結合で導出する枠組みを提案する。その過程で、単一種推定値を多種モデルに固定する際に生じる識別性上の障害（和 vs 積）を定量的に診断し、中立安定系でMSFが成立する条件を論じる。**

- 肯定形の「MSF数値を出した」ではなく、**「レジーム別MSF計算の枠組み＋その成立条件・限界の解明」**を主軸にする。
- 数値（NLM=691.6, LM=9127.2 千トン/年）は**看板にしない**（LM値は自己診断で「長期外挿アーティファクト」と判定済み、かつ過剰適合パラメータ上の値）。

---

## 2. 対象と手法（提案するパイプライン）

- **系**: 被食者=マイワシ(x1)＋ウルメイワシ(x2) / 捕食者=ブリ(y1)＋サワラ(y2)、太平洋系群。
- **レジーム分割**: NLM 2006–2016 / LM 2017–2024。
- **モデル**: 環境収容力なし（自己密度依存項なし）の Lotka–Volterra 型 12変数ODE（`capacity_ry`）。正規化空間（各種を全期間平均で除す）で推定。
- **提案パイプライン（3段）**:
  1. **Catch-MSY**（連続時間 Schaefer）で被食者の増殖率 `r_x` と捕食者の変換効率和 `c1+d1, c2+d2` を漁獲量時系列から推定。
  2. それらを固定し、**残り8自由変数**を制約付きODE推定（`model_constrained.py`）。
  3. 推定パラメータに定数漁獲圧ベクトル `f` を注入し、期間T平均漁獲量を最大化する `f*` を探索 → **レジーム別MSF**。

## 3. 提案の途中で見つかった構造的障害（＝発表の科学的中身）

### 障害A: Catch-MSYはブーム・バスト種を表現できない

- **マイワシ・カタクチイワシ**は環境駆動のレジームシフト種で、Schaeferモデルの**定常生産関数仮定を構造的に破る**。
- マイワシ: 標準の終端枯渇度レンジ [0.01,0.4]（終端 catch/max=0.20 に対応）では生存サンプル **n=0**。高レンジ [0.6,0.95]（＝資源がKの60–95%＝ほぼ満杯）を**強制した時のみ**解ける。しかしこれは「漁獲が歴史的最大の20%しかない低調期」という観測と正面から矛盾する**非物理的な事前分布**。→ r_x1=0.90 はアーティファクト。
- カタクチ: 2015打ち切りでは低レンジで一度解けた（n=14）が、2016–2023を補完すると n=0 に逆転（終端/max=0.09）。**打ち切りアーティファクトで、本質的には解けない**。→ ウルメイワシに置換。
- **含意**: パイプラインのステップ1は、緩やかな定常変動種（ウルメ・ブリ・サワラ）でのみ健全に機能する。ブーム・バスト種は別扱いが必要。

> 補足（要旨には出さないが議論用）: 「黒潮大蛇行 LM/NLM（流路の双安定, 月〜数年スケール）」と「マイワシ本体を駆動する多年代レジームシフト（PDO・移行域SST, 数十年スケール）」は別物。因果を大蛇行に帰属するのは未検証で、要旨では因果を弱めた「レジーム分割による比較」表現にする。

### 障害B: 和 `c+d` を固定すると捕食圧 L が非物理的に歪む（識別性の壁）

- 捕食者の式 `dy1 = (−r_y1−f_y1)y1 + C1·L11·x1·y1 + D1·L21·x2·y1` で、**データが識別するのは積 `C1×L11`, `D1×L21`**（x, y は観測量で、各項の係数=積が拘束される）。C1, L11 単体は識別されない。
- 一方 Catch-MSYが出せるのは**和 `c+d`**（＝各餌を最大限食べたときの捕食者の増殖率上限）。**和は、データが決めている積とは別方向の量**。
- `c+d` を固定すると、積を保つために C, D を動かし、その相棒の **L を代償的に歪める**。L は被食者の式にも共有される（`dx2` の `−L21·x2·y1`）ため、**歪みが被食者のフィットに漏れて悪化**する。
- **実測**（自由推定 vs 制約推定）:
  - 積は安定: LM期 `C1×L11` = 0.210 → 0.208、`D2×L22` = 0.369 → 0.367
  - 個々の因子は10–100倍ぶれる: **L21: NLM 0.224 → 1.00**
  - 適合度悪化: 平均NRMSE NLM 0.146→0.364（2.5倍）、LM 0.079→0.120（1.5倍）
- **正規化空間での L の意味**: 各種が平均1に規格化されているため、`L·y` ≈ 被食者の年あたり捕食死亡率（割合）。健全レンジは **L ≈ 0.1–0.3**（捕食死亡10–30%/年）で、必ず `L < r_x`。L21=1.00 は瞬間率1.0/年（≈年63%消失、増殖率 r_x2=0.78 を超過）で**生物学的に破綻**。
- **根本のミスマッチ**: Catch-MSY（単一種）は L を知らず**和しか出せない**。ODEは**積しか識別しない**。この結合の設計思想の根に構造的非識別性がある。period を変えても解消しない。

### 障害C: 中立安定系でMSFが成立する条件は未定式化

- 自己密度依存がないと、被食者の収量が漁獲率 f の1次（単調）になり、内点MSFが立たない（documented mechanism）。
- 観測: **NLMマイワシ=山型（内点ピーク f≈0.30, MSFあり）** vs **LMマイワシ=単調（f=0.95まで, MSFなし）**。
- 未解決の緊張: 密度依存がないのに **なぜNLMは内点ピークを持つのか**。内点を作る候補は (1) 捕食カップリング（相互作用が実効的制限を与える）、(2) 有限の積分期間T、(3) 非減少制約 `B(T)≥0.9B(0)`。**どれが効いているかは未切り分け**。
- → 「中立安定＋レジーム下でMSFが成立する条件」は**現時点で研究上の問い**。`diagnose_iwashi.py` で各要因をon/offする要因分解で答えられる見込み（合成データ不要、既存コード拡張）。

## 4. 発表のストーリーライン（要旨の段落構成案）

1. **背景**: 黒潮レジームと日本近海の資源変動。単一レジーム前提のMSFでなく、レジーム別MSFを問う意義。
2. **提案**: 単一種Catch-MSY ＋ 多種ODE ＋ 定数漁獲圧最適化を結合した、レジーム別MSF計算の枠組み。
3. **知見1（障害A）**: この枠組みは緩やかな定常変動種で機能し、ブーム・バスト種（マイワシ・カタクチ）ではCatch-MSYの定常性が破れて適用外——事前分布の非物理性として定量化。
4. **知見2（障害B）**: 単一種推定値（和 c+d）を多種ODEに固定すると、ODEが識別する量（積 c×L）との不一致から捕食圧Lが非物理化する識別性の壁を発見・定量化。
5. **知見3（障害C）**: 中立安定系では内点MSFの成立が限定的で、その成立条件（捕食カップリング/有限T/制約の寄与）を論じる。
6. **結論**: レジーム別MSF計算の枠組みを提示するとともに、単一種指標を多種力学へ結合する際の識別性・定常性の限界を明らかにした。
